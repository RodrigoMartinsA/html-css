Links Externos

Usamos a tag <a> para introduzir os links externos ou interno mesmo.

Entra no Google

Usando Target

  1. _blank
    • vai abrir o link em uma nova janela em branco.
      • Google com _blank
  2. _self
    • Vai abrir o link na janela ou frame atual(Padrão).
      • Google com _self
  3. _top
    • vai desfazer todos os frames e abrir o destino no navegador completo.
      • Google com _top
  4. _parent
    • Similar ao uso do _top em uma referência à janela mãe.
      • Google com _parent

Usando Rel

  1. Next
    • Indica que p link é para a próxima parte do documento atual.
      • Google com next
  2. Prev
    • Indica que o link é para a parte anterior do documento atual.
      • Google com Prev
  3. Author
    • Indica que é um link para o site do autor do artigo atual.
      • Google com Author
  4. External
    • Indica que é um link para outro site que não faz parte do site atual.
      • Google com External
  5. Nofollow
    • Indica que pe um link para um site não endossado, como um link pago.
      • Google com Nofollow
